# Notebox

A light-weght web to take notes

## Setup & Installation

Make sure you have the latest version of Python installed.

```bash
git clone <repo-url>
```

```bash
pip install -r requirements.txt
```

## Running The App

```bash
python main.py
```
Or
```bash
./main.py

## Viewing The App

Go to `http://127.0.0.1:8000`
